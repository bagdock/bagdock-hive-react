# Changelog

## 0.1.0 (2026-04-04)

### Features

- Initial public release extracted from the Bagdock monorepo.
- `<HiveProvider>` — configures the Hive client and theming for all child components.
- `<HiveChat>` — embeddable AI chat widget.
- `<HiveBooking>` — unit booking flow widget.
- `<HiveAccess>` — gate access code display widget.
- `useHive()` and `useHiveClient()` hooks for custom integrations.
- Stripe/Clerk-style `appearance` prop for theming with CSS variables.
- Dark mode support via `theme: 'dark'`.
